Lab 6 – Interactive Mapping Project

Project Overview

This lab focuses on creating an interactive map using Python and web mapping tools. The goal was to visualize meaningful geographic locations and design a map that communicates both data and personal context.

Interactive Map

Below is my interactive hometown map. It includes custom styling, markers, and meaningful locations.

Technical Implementation

Tools Used

  • Python
  • Folium
  • Geocoding
  • HTML + CSS integration

How the Map Works

The map was generated using Python and exported as an HTML file. It uses geographic coordinates to place markers and visualize specific locations. The interactive features allow users to zoom, pan, and click markers for more information.

Reflection

What I Learned

This lab strengthened my understanding of how geographic data can be transformed into interactive visualizations. I learned how latitude and longitude coordinates translate into real-world mapping systems.

Challenges

One challenge was properly formatting and importing location data. Debugging geocoding errors required careful checking of data structure and syntax.

Design Decisions

I focused on keeping the design clean and readable. Color, marker placement, and zoom level were intentionally chosen to create clarity while maintaining visual appeal.
