Design Inspiration

The basemap design was inspired by one of my favorite films, Pride and Prejudice, directed by Joe Wright. While browsing Pinterest, I found an image that included the color palette shown below. I selected the fourth color, “Muted Ochre,” as the base for the map and built the rest of the design around it. The muted ochre gave the map a realistic rustic look, which is what I was going for. The slate blue grey was used for bodies of water, while grey and cream tones were applied to major roads and place labels.

Cartographic Challenges

I wanted the map to be legible; I chose colors and labels that work across multiple scales. Colors that look fine for a city might be overwhelming for a country map; this was quite a challenge for me. I also tried to make sure that the labels didn’t overlap each other, which could make it difficult for the user to engage with. Overall, there weren’t many challenges involving the map making process, it was actually quite fun.

Technical Challenges

I encountered several coding challenges throughout this lab, starting with the initial creation of the code. At first, I asked for help from ChatGPT, but despite trying multiple prompts, it wasn’t producing the results I needed. I then remembered that I had experience with an AI tool called Claude from my AI Special Topics class. Claude was better able to understand my code and guide me through the issues I was facing. Not only did it help generate the code, but it also assisted with problems related to my CSV file being in the wrong location. There were many instances when my code wouldn’t run in the terminal, and Claude not only helped me correct these errors but also explained what the code was doing, improving my understanding along the way.
